Passo-a-passo para instalação

1. instalar python (testado com o 3.8.8)
2. instalar anaconda (vai ser o ambiente usado)
3. instalar o visual studio (pelo c++. vc pode instalar o sistema para fazer build com c++ separado tbm)
4. abrir o prompt do anaconda:
conda install -c conda-forge libsndfile
conda install -c conda-forge ffmpeg
pip install ffprobe
pip install numpy==1.20.3 (essa foi a versão com que testei e também a versão atual (06/06/2021))
pip install timidity
pip install chord-extractor
Se estiver no Windows, instalar: https://code.soundsoftware.ac.uk/projects/vamp-plugin-pack
pip install PySimpleGUI
pip install ShazamAPI
pip install beautifulsoup4
pip install pychord

(Para o linux, pode usar também o sudo apt-get para instalar o que foi instalado com o conda)
